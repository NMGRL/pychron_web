Welcome to SampleTracker

SampleTracker is a simple web app for managing samples in a Pychron Database. It is based on Django and deployed with Docker. It is freely available at Github

The following process is used to add a sample to the system.
  1. Add the Principal Investigator. Enter a single name or lastName,firstInitial e.g "Ross,J"
  2. Add the Project. Select the Principal Investigator from step 1. Enter a name for the Project. If ? is used than the project name will be generated from the PI'slast name and the Project's IR number
  3. Add the Sample
